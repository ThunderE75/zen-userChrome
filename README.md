# Zen Customization 

My userChrome.css for my Zen Browser

## Features

1. Pure Black Theme
1. Auto-dim URL Bar, Extensions & Panel Icons
2. Bottom Essentials & Auto-hide
3. Auto-dim Bottom Workspace indicator
4. Custom Dot styled Minimize, Maximize & Close Buttons
